The best way to promote a product line, a type of service, or even a person is through the internet. Before the invention of various website builders like SquareSpace or other "no-code website builders", most of the internet was created and maintained by a large body of documents written using programming languages that help in building the structure and layout help from websites.

A static website is a type of website that does not require any server-side scripting or database interactions. It consists of simple HTML, CSS, and even JavaScript files that are served to the user's browser. To create a static website using HTML and CSS, you will have to consider the following steps:

1. Plan the structure and content of your website.
2. Write HTML code to structure the content and add visual elements such as headings, paragraphs, images, and links.
3. Write CSS code to style your website, control the layout, colors, font styles, and other visual aspects.
4. Combine both HTML and CSS code for each page of your website.
5. Upload the files to a web server or hosting service to make your website accessible to the public.

HTML files are made up of tags that define the structure and content of a web page. Some of the most important tags are the following:

<html>: Is the root tag of an HTML document. All other tags are contained within this tag.

<head>: This tag contains information about the document that is not displayed in the browser window. (title of the page, search engine keywords, other type of metadata).

<body>: This tag contains the content of the HTML document that is displayed in the browser window.

<header>: Contains the main menu or even the logo of a web page.

<main>: This tag contains the main content of the web page.

<section>: This tag represents a unique section of a web page that behaves uniquely from another. It is usually determined by an id, which is complimented by CSS attributes.

To create a navigation bar, you need to write both HTML and CSS code to add not only the structure of how many sections you want to add to your homepage, but also whether you want to add links to different pages within your website. The following code displays the <nav> tag in HTML.

The <nav> tag is used to contain navigation links to other pages or sections, as a way to group a list of links related to the main site navigation. CSS adds functionality and appearance functions to the <nav> tag, like adding hover effects to make the links more interactive.

Meanwhile, the CSS component for the <nav> tag can either be written directly in the HTML file using the <style> tag, or you can link to an external CSS file using the <link> tag in the <head> section of the HTML file.

In either case, the <nav> tag in CSS can contain attributes such as text alignment, background color, display settings, and many more.

To add media such as an image or a video, the <img> tag can have html attributes such as: the attribute 'src' that specifies the location of the image file, and the 'alt' attribute that provides a description of the image. Additionally, the CSS code can add attributes such as: width, length, border-radius, and many more.

Overall, the <img> tag can be enhanced by CSS components in order to make them more visually appealing by changing the borders, shadows, filters, effects, or even make it responsive to the size of the user's screen.

PyScript is a Python library that allows you to execute Python scripts from within a web page using only HTML code, as it serves as a viable alternative to what would usually be handled by the programming language of Javascript, which enables the creation of much more complex web pages that interact with databases, generate charts and graphs, and even perform other server-side tasks within dynamic websites.

PyScript is able to import common Python modules and run Python code seamlessly within an HTML file just by linking to - pyscript.net - and importing relevant library packages. Consecutively sharing extensible components that would be only available through Python.

NOTE: PyScript has some limitations as it may not be able to execute ALL Python libraries but on the other hand it has the capability to run many popular packages within the scientific stack (such as numpy, pandas, scikit-learn, matplotlib, and more).
